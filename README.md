# EZ4H
Allow Minecraft:JE(1.12.2) connect to Minecraft:BE(1.16.20) servers!  
## Features
- [x] Login  
- [x] Chat  
- [x] Command  
- [ ] Xbox Auth  
- [X] Chunks  
- [X] Move  
- [ ] Players  
- [ ] Entitys  
- [ ] Attack  
- [ ] Block Interact  
- [ ] Titles and other Messages
